### 1. How to Install / Upgrade
To install this chart (with all subcharts):
  helm install {{ .Release.Name }} ./ecommerce -n {{ .Release.Namespace }}

To upgrade after making changes:
  helm upgrade {{ .Release.Name }} ./ecommerce -n {{ .Release.Namespace }}

If you want to install into a different namespace:
  kubectl create namespace my-namespace
  helm install {{ .Release.Name }} ./ecommerce -n my-namespace

---

### 2. How to Verify Pods are Running
Check the status of all pods:
  kubectl get pods -n {{ .Release.Namespace }}

You should see pods for the parent chart (ecommerce) and all subcharts 
(auth, cart, payment, frontend, etc.) in the `Running` state.

For detailed pod logs:
  kubectl logs <pod-name> -n {{ .Release.Namespace }}
